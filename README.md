# F1 Hub

A responsive, dynamic Formula 1 website displaying real-time data for driver standings, race schedules, and grid information. Features a modern, high-performance aesthetic inspired by F1 racing, complete with a car gallery and smooth animations.

## Features
- **Live Driver Standings**: Up-to-date points and rankings.
- **Race Schedule**: Explore past and upcoming races.
- **Car Gallery**: High-quality views of F1 vehicles.
- **Futuristic UI**: Dynamic animations and sleek styling crafted with HTML, CSS, and modern JavaScript.

## Setup
To run this project locally:
1. Clone the repository.
2. Start a local server (e.g., `python -m http.server 8000` or via VS Code Live Server).
3. Open `http://localhost:8000` in your web browser.
